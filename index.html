How to Use

1

Download the Workflow

Click the download button above to get the latest version of BPM_Automation.workflow.zip from our GitHub releases.

2

Install the Workflow

Double-click the BPM_Automation.workflow file on your Mac. The Quick Action Installer will open and guide you through the installation process.

3

Assign a Keyboard Shortcut

  1. Open System Settings (or System Preferences)
  2. Go to Keyboard > Keyboard Shortcuts
  3. Select Services (or Quick Actions)
  4. Find BPM_Automation in the list under General
  5. Assign a custom keyboard shortcut (e.g., ⌥⌘B)
4

Start Using It

Open Apple Music, select one or more tracks, and press your keyboard shortcut. Enter the desired BPM when prompted, and the script will handle the rest!

Requirements & Troubleshooting

⚙️ System Requirements

  • macOS with Apple Music app
  • Automator (included with macOS)

🔧 Permissions

Make sure you have granted the necessary permissions:

  • Go to System Settings > Security & Privacy > Privacy > Automation
  • Grant permissions for both Automator and Apple Music

⚠️ Important Notes

  • The script only works when tracks are selected in Apple Music
  • Only positive integer BPM values are accepted
  • The track name must have a dash separator (e.g., "120 - Song Title")
